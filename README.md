# Portfólio Acadêmico

Este é um projeto de portfólio acadêmico desenvolvido para apresentar projetos, habilidades e informações pessoais relacionadas ao curso de Engenharia de Software.

## 📚 Sobre o Projeto

O objetivo deste portfólio é reunir projetos acadêmicos, experiências e conhecimentos adquiridos durante a graduação.

## 🚀 Tecnologias Utilizadas

- Python
- Flask
- HTML5
- CSS3
- Git e GitHub
